## 简易的h5壳子，方便后续开发

> 提示

- 使用[VARLET组件库](https://varlet.gitee.io/varlet-ui/#/zh-CN/index)
- 切换主题，使用scss来搭配varlet组件库的StyleProvider 来进行全局的替换 详细看 `layout/header/HeadBar` 文件
- 简易明了，本人是个小菜鸡，没有什么高端写法，简陋的AnyScript
- 别看了没有了，就是个初始化壳子
